# 粒子系统修改验证清单

## 🔍 验证步骤

### 1. 清除浏览器缓存
- [ ] 按 `Ctrl + Shift + Delete` 清除缓存
- [ ] 或使用无痕模式打开
- [ ] 或按 `Ctrl + F5` 强制刷新

### 2. 访问登录页
- [ ] 打开 http://localhost:5174/login
- [ ] 观察粒子是否从四周缓慢飞入
- [ ] 粒子应该从屏幕外200-500像素处飞入

### 3. 检查粒子运动速度
- [ ] 粒子转动速度应该很慢(频率0.1-0.3)
- [ ] 粒子沿方程运动速度应该很慢(0.006)
- [ ] 不应该感到眼花缭乱

### 4. 检查粒子数量
- [ ] 1920x1080屏幕应该有约78个粒子
- [ ] 粒子密度应该适中,不稀疏也不拥挤

### 5. 检查生命周期
- [ ] 粒子应该有15-25秒的生命周期
- [ ] 粒子应该渐隐消失
- [ ] 新粒子应该从上方缓慢飘入

### 6. 检查数学方程
- [ ] 应该能看到12种不同的数学方程
- [ ] 包括蝴蝶、心形、星形等高阶方程
- [ ] 每次刷新应该随机选择不同方程

### 7. 登录进入首页
- [ ] 点击登录按钮
- [ ] 观察路由切换到首页
- [ ] 粒子应该重新从四周缓慢飞入
- [ ] 应该选择新的数学方程

## 🔧 故障排除

### 问题1: 粒子没有缓慢飞入
**解决方案**:
1. 清除浏览器缓存
2. 强制刷新页面(Ctrl + F5)
3. 检查控制台是否有错误

### 问题2: 运动速度仍然很快
**解决方案**:
1. 确认使用的是新端口(5174)
2. 检查是否加载了最新代码
3. 查看控制台Network面板

### 问题3: 路由切换无效果
**解决方案**:
1. 确保在登录页和首页之间切换
2. 检查路由监听是否生效
3. 查看控制台日志

### 问题4: 方程没有变化
**解决方案**:
1. 刷新页面多次
2. 检查随机选择逻辑
3. 查看控制台输出
