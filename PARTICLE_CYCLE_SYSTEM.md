# 粒子自动循环系统说明

## 🔄 循环流程

粒子系统现在实现了完整的自动循环效果,流程如下:

```
空闲(3秒) → 聚合(2秒) → 爆开 → 吸附(5秒) → 空闲(3秒) → ...
```

### 详细时间线

| 阶段 | 持续时间 | 描述 |
|------|----------|------|
| 空闲 | 3秒 | 粒子自由运动,等待下一轮循环 |
| 聚合 | 2秒 | 所有粒子向随机中心点聚集 |
| 爆开 | 瞬间 | 粒子爆发式向四周扩散 |
| 吸附 | 5秒 | 粒子缓慢向屏幕中心移动 |

## 🎯 各阶段详解

### 1. 空闲阶段 (IDLE)
- **持续时间**: 3秒
- **粒子行为**: 自由运动,受轻微随机波动影响
- **触发条件**: 组件挂载后3秒,或吸附阶段结束后3秒
- **状态**: `cyclePhase = 'idle'`

### 2. 聚合阶段 (AGGREGATING)
- **持续时间**: 2秒
- **粒子行为**: 向随机中心点快速聚集
- **聚合中心**: 屏幕30%-70%范围内的随机位置
- **吸引力**: force = 0.15(强力吸引)
- **速度衰减**: 接近中心时速度衰减到0.8倍
- **状态**: `cyclePhase = 'aggregating'`

### 3. 爆开阶段 (DISPERSING)
- **持续时间**: 瞬间
- **粒子行为**:
  - 原有粒子随机方向发散(速度3-8像素/帧)
  - 生成50个新粒子环形爆发(速度6-14像素/帧)
- **视觉效果**: 震撼的爆发效果
- **状态**: `cyclePhase = 'dispersing'`

### 4. 吸附阶段 (ATTRACTING)
- **持续时间**: 5秒
- **粒子行为**: 缓慢向屏幕中心移动
- **吸引力**: force = 0.02(温和吸引)
- **触发距离**: 距离中心50像素以上
- **状态**: `cyclePhase = 'attracting'`

## 🔧 技术实现

### 状态管理
```typescript
let autoCycleEnabled = true  // 是否启用自动循环
let cyclePhase: 'idle' | 'aggregating' | 'dispersing' | 'attracting' = 'idle'
let cycleTimer: number | undefined

const AGGREGATE_DURATION = 2000  // 聚合持续时间
const ATTRACT_DURATION = 5000    // 吸附持续时间
const IDLE_DURATION = 3000       // 空闲时间
```

### 核心函数

#### startAutoAggregate()
- 检查是否可以开始聚合(未在鼠标交互中)
- 随机选择聚合中心点
- 保存粒子原始位置
- 设置聚合状态
- 2秒后触发发散

#### disperseParticles()
- 设置发散状态
- 原有粒子随机发散
- 生成爆发粒子
- 5秒后触发吸附

#### startAttracting()
- 设置吸附状态
- 粒子缓慢向中心移动
- 5秒后回到空闲状态
- 3秒后开始新一轮循环

### 动画循环集成
```typescript
// 在animate函数中根据cyclePhase执行不同逻辑
if (cyclePhase === 'attracting' && p.life === Infinity) {
  // 吸附逻辑
  const centerX = canvas.width / 2
  const centerY = canvas.height / 2
  // ... 向中心移动
}
```

## 🎮 用户交互优先级

当用户进行鼠标交互时,自动循环会被暂停:

1. **鼠标移动**: 暂停自动循环,粒子跟随鼠标
2. **鼠标离开**: 恢复自动循环
3. **长按触发**: 手动聚合,覆盖自动循环
4. **单击**: 爆发效果,不影响循环状态

## 📊 性能优化

### 定时器管理
- 使用setTimeout精确控制各阶段时长
- 组件卸载时清理所有定时器
- 设置`autoCycleEnabled = false`停止循环

### 条件判断
- 检查`autoCycleEnabled`避免不必要的计算
- 检查`mouse.isActive`优先响应用户交互
- 检查`p.life === Infinity`只处理永久粒子

### 内存管理
- 及时清理临时粒子(生命周期150帧)
- 避免创建过多定时器
- 组件卸载时完全清理

## 🎨 视觉效果

### 聚合阶段
- 粒子快速聚集,形成密集中心
- 连接线密集,产生网络效果
- 深色模式下发光明显

### 爆开阶段
- 环形爆发,360度均匀分布
- 速度递增,产生扩散效果
- 粒子大小随机,增加层次感

### 吸附阶段
- 缓慢移动,优雅过渡
- 向中心聚集,但不完全重合
- 保持一定分散度

### 空闲阶段
- 自由运动,自然波动
- 连接线动态变化
- 等待下一轮循环

## 🔄 循环周期计算

完整循环周期 = 空闲(3s) + 聚合(2s) + 吸附(5s) = **10秒**

每小时循环次数 = 3600s / 10s = **360次**

## 🚀 使用建议

1. **性能考虑**: 移动设备可适当增加空闲时间
2. **视觉效果**: 深色模式下效果更佳
3. **交互平衡**: 自动循环与用户交互和谐共存
4. **自定义**: 可调整各阶段时长适应不同场景

## 🎯 最佳实践

- 登录页: 默认配置,营造动态感
- 首页: 可增加空闲时间到5秒
- 兴趣配置页: 保持默认,增强引导感
